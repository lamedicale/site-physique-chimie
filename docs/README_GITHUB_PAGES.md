# Site prêt pour GitHub Pages

Ce dossier contient une version prête à publier du site.

## Publication conseillée

1. Créer un dépôt GitHub public.
2. Envoyer le dossier `docs/` à la racine du dépôt.
3. Aller dans `Settings → Pages`.
4. Choisir :
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/docs`
5. Enregistrer.

La page d'accueil est `docs/index.html`.
